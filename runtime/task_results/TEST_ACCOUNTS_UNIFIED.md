## 云端同步边界

云端同步会写入服务器测试数据库，属于云端数据修改。执行前需要确认：

1. 是否允许覆盖或补齐云端测试账号。
2. 是否允许重启云端 API 服务。
3. 是否只同步测试账号，不清理订单数据。
