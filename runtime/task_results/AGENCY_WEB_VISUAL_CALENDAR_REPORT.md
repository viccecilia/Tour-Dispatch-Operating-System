# 旅行社 Web 可视化日历报告

## 修改范围
- 文件：`frontend/src/pages/AgencyPortalPage.tsx`
- 范围：仅旅行社 Web 后台日历板块
- 未改动：车公司端小程序、旅行社小程序、总控管理端、后端接口、数据库、云端部署

## 功能
- 将原来的日期筛选列表升级为月历视图。
- 后续调整为横向订单日历条：纵轴为订单编号，横轴为日期窗口。
- 纵轴订单编号优先使用现有 `oid`，格式为旅行社/来源代码 + 日期 + 当日流水，例如 `A260621-0001-TMP`。
- 已接单订单显示扩展编号：基础订单号 + 车公司代码缩写 + 接单日期 YYMMDD + 车公司流水，例如 `A260621-0001-TMP-CC260531-0001`。
- 已分配司机后继续追加司机名字缩写，显示规则为：`基础订单号-车公司段-司机缩写`。
- 订单条支持跨日期显示，适合直接查看包车占用情况。
- 月历按 7 列展示完整月份，并包含前后月补位日期。
- 每个日期格显示当天订单数量、前 3 条订单摘要和更多订单提示。
- 状态颜色接入：
  - 发布中 / 竞拍中
  - 已接单 / 已派车
  - 流拍 / 未成交
  - 待结算 / 请款中
  - 已完成 / 已结算
- 右侧保留当天行程列表、日期跳转和订单/人数统计。
- 如果当前日期所在月份没有订单，自动展示距离当前日期最近的有订单月份，避免初始空白。
