# WX-DISPATCH-R017：Driver Mobile Experience

## 本轮结果
- 小程序司机端改为手机优先布局。
- 当前任务突出显示。
- 一键下一步大按钮。
- 提交前二次确认，防误触。
- 提交中锁定按钮，防重复提交。
- 弱网/离线提示。
- 离线读取上次同步缓存。
- 调度端 Driver Monitor 升级为 KPI、任务卡片、状态时间线和最新报备。

## 验证结果
- `python scripts/verify_driver_api.py`：通过。
- `npm.cmd run build`：通过。
- `npm.cmd run lint`：通过。
- 浏览器检查 `#driver-monitor`：司机执行监控、状态时间线、最新司机报备均可见。

## 风险
- 小程序真机预览未由命令行自动执行，需要人工在微信开发者工具验证。
- 离线缓存只读，不支持离线报备后联网自动补传。
- 本轮不做地图、不做照片上传复杂流。

## 下一轮建议
- 做司机端正式身份绑定或轻量登录，替换手动 driver_id。
- 如确实需要离线报备，下一轮做本地队列和联网后补传。
