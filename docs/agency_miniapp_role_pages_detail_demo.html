管理端说明

管理端不是录单主力，而是看 100 单的整体风险、额度、人员效率和异常处理进度。

页面应包含的能力

    和 Web 端、订单大厅的联动

    • Web 端适合批量导入、批量改价、批量筛选；小程序端适合待办、确认、沟通、现场反馈。
    • 客服在小程序补齐资料或确认上架后，订单同步进入订单大厅，展示起拍价、一口价、落拍时间。
    • 订单被车公司接走后，小程序按权限解锁车公司负责人、司机电话、微信和服务进度。
    • 财务完成付款后，该订单占用的保证金额度自动释放，管理端和客服端同步看到额度恢复。
