急单规则

不是低价竞拍
优先救到

急单先看 ETA、车辆真实可用、司机状态和历史履约，价格排在后面。

允许加价

急单可高于标准价，尤其是深夜、雨雪、机场排队和误机风险订单。

责任归因

原车公司、旅行社、旅客、平台或不可抗力责任会影响差价承担和扣分。

订单大厅核心规则

平台市场层
位置

订单大厅在总控平台市场层，不属于旅行社或车公司任何一方。

可见性

旅行社只看自己的订单；车公司看公共池中符合资质等级的订单。

报价

报价必须在平台允许区间内，保护底价不对车公司公开。

定标

初期人工定标，后期可按综合评分自动推荐，不默认最低价中标。

风控

保证金、授信额度、客诉、取消、迟到、恶意低价都会影响可接订单范围。
