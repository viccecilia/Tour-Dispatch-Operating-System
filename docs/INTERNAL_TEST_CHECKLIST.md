# Internal Test Checklist

## 基础环境

- [ ] 后端 `/api/ping` 可访问
- [ ] Web 管理后台可打开
- [ ] 司机小程序可打开
- [ ] 调度小程序可打开
- [ ] 微信后台已配置 request/upload/download 合法域名
- [ ] HTTPS 证书有效

## 登录

- [ ] Web 管理员账号可登录
- [ ] 调度小程序账号可登录
- [ ] 司机小程序账号可登录
- [ ] 运行管理账号可登录
- [ ] 非授权微信登录已绑定账号会被拒绝
- [ ] `zongzou` 超级微信测试号可用于内部测试

## 订单与派车

- [ ] 调度小程序可粘贴订单
- [ ] 订单可解析成草稿
- [ ] 草稿可纠错
- [ ] 草稿可确认入库
- [ ] Web 后台能看到订单
- [ ] 调度端可选择订单、司机、车辆
- [ ] 一键派车可写入 assignments
- [ ] 司机端可收到订单
- [ ] 司机确认接单后，调度端状态可见

## 司机执行

- [ ] 点呼出库可提交
- [ ] 到达上车点可提交
- [ ] 完成订单可提交
- [ ] 入库点呼可提交
- [ ] 司机端不显示订单销售价格
- [ ] 司机位置上报 API 可用

## 财务与费用

- [ ] 司机可提交垫付费用
- [ ] 司机可提交代收费用
- [ ] 财务台账可看到完成订单
- [ ] 财务台账可看到司机费用
- [ ] 只有 admin 可访问财务数据

## 备份恢复

- [ ] `python scripts/backup_trial_db.py` 可生成备份
- [ ] 备份文件在 `runtime/backups/trial/`
- [ ] `python scripts/restore_trial_db.py <backup>` 可恢复
- [ ] 恢复后可以重新登录并读取数据

## 真机测试

- [ ] 体验成员可打开体验版
- [ ] 手机非 Wi-Fi 状态可访问 API
- [ ] 调度端录单不卡顿
- [ ] 司机端任务页不卡顿
- [ ] Web 后台与小程序数据同步
