# Framework.xlsx 可复用内容检查

源文件：

## Sheet 概览

- `Main Table`：订单主表样式、订单号规则、字段定义示例。
- `drivers-data`：司机资源库，可迁移司机姓名、司机代码、语言、营业所、电话。
- `car-data`：车辆资源库，可迁移车牌、车辆简码、车型代码、车种、颜色、雪胎。
- `location-data`：地点标准名和别名，已适合 parser 地点归一化继续使用。
- `Order Notes-data`：订单备注代码 `D` / `X`，可继续用于订单号来源和订单来源识别。
- `Order Type-data`：日期格式、订单类型关键词、车型关键词和车型代码规则。
- `Sample`：真实送迎订单样本文本，可继续作为 parser 回归样本。
