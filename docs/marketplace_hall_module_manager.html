当前确认规则

前期版本
一个订单大厅

不再分标准单、急单、回程单三个大厅。全部进入同一个大厅，用标签和来源区分。

拍卖价 + 一口价

车公司每次出价默认比当前最低价低 20 元；任一合格车公司可按一口价直接锁单。

车公司外放订单

车公司订单过量、车辆故障、司机不足时，可以把多余订单外放到大厅，让其他车公司竞拍承接。

成交后解锁

成交后解锁客人联系方式、发单方负责人电话/微信、车公司负责人电话/微信，并进入待分配订单。

车公司外放订单流程

同行消化
1车公司发现订单过量、车辆故障、司机不足、临时无法消化。
2在车公司配单/待分配页面点击“放到订单大厅”，填写起拍价、一口价、落拍时间，确认后发布。
3订单进入同一个订单大厅，来源标记为“车公司外放”，其他车公司按拍卖价或一口价承接。
4落拍或一口价成交后，订单进入承接车公司“待分配订单”，来源标记为“大厅拍得/同行外放”。
5平台保留原承运方、承接方、发单方、通知状态和责任边界，方便后续客诉仲裁。
